The Data USed In Analyses

In order to evaluate the detection system, two independent data sets, each comprising 100 low intensity Embolic Signal, 100 various type of artifacts and 100 Doppler Speckle were used. The total 200 artifacts were created artifically during patient recordings by tapping the probe, by speech or coughing during routine patient recordings

  • After applying the discrete wavelet transform to the data, several parameters were evaluated.
  • The threshold values used for both data sets were optimized using the first data set.
