Getting Started With Project

1How Can I Execute Project?
Download and install GNU Octave free software which is featuring a high-level programming language.
2Try Sample Data Entry
Download the project from Projects section and open it from Octave, After download data from the same section and move sample data to the directory where project is remaining.
3See The Results Of Execution
Execute Command " pkg load signal " before executing the project, After execute project choose the data sample format with non capital letters and finally choose the name of sample data.
